Dorm Room Crochets

Beanie

This purple ribbed beanie was made only doing triple double crochets. You can use any yarn size and its matching hook size. I followed a youtube tutorial by @VivCrochets on Youtube where she shows a detailed step by step on how to crochet this beanie.

Heart Tray

This is a heart-shaped tray that can be used to hold earrings, braclets, etc. You can use any yarn size and its matching hook size. I made this by following a tutorial by @mahumcrochets on Youtube.

Mat

This is a simple white mat with flowers around the edge. This can be placed on your table for decor. For the rectangular section, I simply did triple double crochets. For the flowers, I followed @HopefulTurns Youtube Tutorial "How to Crochet a Simple Flower".
